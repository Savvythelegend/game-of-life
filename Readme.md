# Project structure

Below is the top-level layout of the repository. Files and folders shown reflect
the `rplife` package and project metadata.

```text
rplife/
├── rplife/
│   ├── __init__.py
│   ├── __main__.py
│   ├── cli.py
│   ├── grid.py
│   ├── patterns.py
│   ├── patterns.toml
│   └── views.py
├── README.md
└── pyproject.toml
```

Notes:
- The `rplife/` directory contains the package source.
- `patterns.toml` stores predefined patterns used by the application.
- `pyproject.toml` contains build and dependency configuration.
